# Memory Weave

**Memory Weave** (also known as *Gnyapakam*) is a boutique handcrafted gift store specializing in preserving your most precious memories through beautiful, custom-made weaves and artistic creations.

## 🌟 The Brand
"Memory Weave" is built on the philosophy that memories are the threads that weave our stories together. We provide high-quality, artisanal products that serve as timeless keepsakes for life's most meaningful moments.

## 🚀 Key Features
- **Curated Collection**: Handpicked gift items designed with care.
- **Customization**: Personalized touches for each memory.
- **Modern User Experience**: A fast, responsive React-based storefront.
- **Premium Aesthetics**: Minimalist, elegant design that focuses on the art of memory.

## 🛠️ Technology Stack
- **Frontend**: React + TypeScript
- **Styling**: Tailwind CSS
- **Framework**: Vite
- **UI Components**: Shadcn UI
- **Icons**: Lucide React
- **Routing**: React Router DOM

## 🛠️ Local Development

To run the project locally, follow these steps:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Vinaypenke01/TheOrnateee.git
    ```
2.  **Navigate to the project directory**:
    ```bash
    cd TheOrnateee
    ```
3.  **Install dependencies**:
    ```bash
    npm install
    ```
4.  **Start the development server**:
    ```bash
    npm run dev
    ```
5.  **Build for production**:
    ```bash
    npm run build
    ```
